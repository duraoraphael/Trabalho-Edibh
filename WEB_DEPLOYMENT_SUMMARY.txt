================================================================================
FAQ - DÚVIDAS COMUNS
================================================================================

P: Quanto custa?
R: Railway oferece $5/mês de crédito grátis. Para apps pequenos, dura ~3 meses.

P: Dados são perdidos ao desligar?
R: Sim! JSON local é deletado. Para produção, configure PostgreSQL gratuito.

P: Posso usar meu domínio próprio?
R: Sim! Railroad permite domínios customizados. Compre em Namecheap/GoDaddy.

P: Como atualizar código?
R: Git push para main. Railway faz deploy automático em 2 minutos.

P: Preciso de banco de dados?
R: JSON funciona para teste. Para produção, use PostgreSQL (gratuito no Railway).

P: App está lento?
R: Normal em plano gratuito. Upgrade para plano pago conforme crescer.
